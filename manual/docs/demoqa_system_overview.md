## System Description
The DemoQA Automation Practice Form is a public web application designed to simulate a real-world user registration form.
It contains multiple input types such as text fields, radio buttons, checkboxes, date picker, file upload, and submit functionality.

## URL
https://demoqa.com/automation-practice-form

## Purpose of Testing
The purpose of this testing is to validate form behavior, input validations, error handling, and successful form submission from a QA manual perspective.
