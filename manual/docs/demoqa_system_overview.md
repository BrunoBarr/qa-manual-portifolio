## System Description
The DemoQA Automation Practice Form is a public web application designed to simulate a real-world user registration form.
It includes multiple input types such as text fields, radio buttons, checkboxes, date picker, file upload, and form submission.

## URL
https://demoqa.com/automation-practice-form

## Target Users
- QA engineers and testers practicing form validation
- Developers validating front-end behavior

## Main Features
- Text input fields
- Radio button selection
- Checkbox selection
- Date picker
- File upload
- Form submission

## Purpose of Testing
Validate form behavior, input validations, error handling, and successful form submission from a manual QA perspective.

## Test Environment
- Platform: Web  
- Browser: Chromium-based (Desktop)
